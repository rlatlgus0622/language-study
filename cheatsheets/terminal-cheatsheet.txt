[ 터미널 명령어 모음 ]


1. pwd (Print Working Directory)
- 현재 내가 터미널상에서 어느 폴더에 들어와 있는지 전체 경로를 보여줌
(옵션 없음): 현재 위치 주소 출력


2. ls (List) 
- 현재 위치한 폴더 안에 무엇이 들어있는지 목록을 보여줌
    ls -a: 숨겨진 파일(파일명이 .으로 시작하는 .gitignore 등)까지 모두 표시
    ls -l: 파일의 권한, 소유자, 용량, 수정 날짜 등 상세 정보 표시
    ls -al: (가장 많이 씀) 숨김 파일 포함 상세 정보를 리스트 형태로 표시
    ls -R: 하위 폴더 내부의 내용까지 모두 펼쳐서 표시


3. cd (Change Directory)
- 폴더 위치를 이동할 때 사용
    cd [폴더명]: 해당 폴더 안으로 진입
    cd ..: 현재 폴더의 상위(밖) 폴더로 한 단계 이동
    cd ~: 내 컴퓨터의 최상위 홈(Home) 디렉토리로 이동
    cd -: 바로 이전에 있었던 폴더로 다시 이동
    cd /: Mac 시스템의 루트(최상단) 경로로 이동


4. mkdir (Make Directory)
- 새로운 폴더를 생성
    mkdir [폴더명]: 현재 위치에 새 폴더 생성
    mkdir -p [폴더/하위폴더]: 하위 폴더까지 계층 구조로 한 번에 생성 (예: mkdir -p c/study)


5. touch (Touch)
- 내용이 없는 빈 파일을 생성
    touch [파일명]: 해당 이름의 빈 파일 생성 (예: touch README.md)

6. cp (Copy)
- 파일이나 폴더를 복사
    cp [원본파일명] [대상파일명]: 파일을 복사해서 새로운 파일 생성
    cp -r [원본폴더] [대상폴더]: 폴더 전체(내부 파일 포함)를 복사


7. mv (Move)
- 파일을 이동시키거나 이름을 변경
    mv [원본파일명] [이동경로]: 파일을 해당 경로로 이동
    mv [기존이름] [새이름]: 같은 위치에서 이름만 변경 (파일 이름 바꾸기)


8. rm (Remove)
- 파일이나 폴더를 삭제합니다. (주의: 휴지통을 거치지 않고 즉시 삭제됩니다)
    rm [파일명]: 파일 삭제
    rm -f [파일명]: 질문하지 않고 강제로 삭제
    rm -r [폴더명]: 폴더와 그 안의 모든 내용을 포함하여 삭제
    rm -rf [폴더명]: (주의!) 폴더 전체를 묻지도 따지지도 않고 강제로 삭제


9. cat (Concatenate)
파일의 내용을 터미널 화면에 바로 출력
    cat [파일명]: 파일 안의 텍스트를 터미널 창에서 읽기


10. clear (Clear)
- 터미널 화면에 가득 찬 텍스트를 깨끗하게 비우고 맨 위로 올림
(단축키: Command + K)


11. history (History)
- 지금까지 입력했던 모든 명령어 목록을 번호와 함께 보여줌
    history [숫자]: 최근 입력한 명령어 중 해당 숫자만큼만 표시
    ! [번호]: history 목록의 해당 번호 명령어를 다시 실행


12. open (Open)
- Mac의 Finder나 특정 프로그램으로 파일/폴더 열기
    open .: 현재 위치를 Finder(탐색기) 창으로 열기
    open [파일명]: 해당 파일을 기본 프로그램으로 실행
    open -a [프로그램명]: 특정 프로그램으로 파일 열기 (예: open -a Xcode main.c)


13. man (Manual)
- 명령어의 사용법(설명서)을 보여줌
    man [명령어]: 해당 명령어의 모든 옵션과 설명 보기 (나가려면 q 입력)


14. sudo (Superuser Do)
- 관리자 권한으로 명령어를 실행 (Mac의 시스템 비밀번호를 입력해야 함)
    sudo [명령어]: 최고 관리자 권한으로 작업 수행



반드시 익혀야 할 터미널 단축키

Tab 키: 파일명이나 폴더명 일부만 적고 Tab을 누르면 자동으로 전체 이름이 완성됩니다. (오타 방지 필수!)
↑ / ↓ (방향키): 이전에 쳤던 명령어들을 위아래로 훑으며 다시 불러올 수 있습니다.
Ctrl + C: 현재 실행 중인 명령어를 강제로 종료하거나, 입력을 취소하고 새 줄로 갑니다.
Ctrl + A: 커서를 줄 맨 앞으로 이동
Ctrl + E: 커서를 줄 맨 뒤로 이동
Ctrl + U: 현재 입력 중인 줄 전체 삭제
