# Markdown 문법 정리

## 1. 제목

```
# 제목 1
## 제목 2
### 제목 3
#### 제목 4
##### 제목 5
###### 제목 6
```

> 제목 1과 제목 2에만 구분선이 생김

---

## 2. 텍스트 강조
```
**굵게**

*기울임*

***굵게 + 기울임***
```

---

## 3. 목록

### 순서가 없는 목록
```
- 항목 1
- 항목 2
    - 하위 항목 (앞에 스페이스 2칸)
```

### 순서가 있는 목록
```
1. 첫번째
2. 두번째
3. 세번째
```

---

## 4. 코드

### 인라인 코드 

**짧은 명령어에 사용**

```
`rm -rf` `ls -al` `cd ..`
```

**결과**
`rm -rf` `ls -al` `cd ..`

### 코드 블록 

**여러 줄 코드에 사용**

````
```
bash
cd 폴더명
ls -al
rm -rf 폴더명
```
````

## 7. 링크 & 이미지

### 링크

```
[텍스트](URL) 형태
```

**결과**
[GitHub](https://github.com)

### 이미지

```
[텍스트](이미지 URL)
```
